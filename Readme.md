# Starter boilerplate Angular and NestJS

## Includes

- Register
- Login with credentails and google
  \_ Reset password
- Manage profile
- Search with meilisearch (API)

## Install dependencies

- Angular (client) && Nest (API)
  `pnpm i`

## Start

- Angular
  `pnpm start`
- NestJS
  `pnpm start:dev`

## Lint

- Angular (client) && Nest (API)
  `pnpm lint`

## Build

- Angular (client) && Nest (API)
  `pnpm build`

NB: I use commit lint(for commit messages)
